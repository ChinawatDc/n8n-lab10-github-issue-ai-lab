# GitHub Issue AI Reviewer – Test Repo

This repository is used to test an automated GitHub Issue Reviewer powered by AI.

## Purpose
- Test GitHub Webhook (issues opened)
- Test AI classification (bug / feature / question)
- Test auto-comment via GitHub API

## How to test
1. Create a new Issue
2. Write a title and description
3. Check the automated AI comment 🤖

> This repository contains no production code.
